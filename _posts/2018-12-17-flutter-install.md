### 创建新应用
- 选择 File>New Flutter Project
- 选择 Flutter application 作为 project 类型, 然后点击 Next
- 输入项目名称 (如 myapp), 然后点击 Next
- 点击 Finish

在项目目录中，您应用程序的代码位于 lib/main.dart
