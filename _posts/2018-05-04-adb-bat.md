### databases.bat


```
rd /s /q databases

::pause
adb shell < cmd.txt
adb pull /mnt/sdcard/databases
adb shell rm -r /mnt/sdcard/databases
```


### cmd.txt

```
su
cp -r /data/data/com.eascs.iretail/databases /mnt/sdcard
exit
exit
````
