### 1、安装[nodejs](https://nodejs.org/en/download/)

在nodejs的安装目录下，新建 node_global和 node_cache 两个文件夹
设置nodejs prefix（全局）和cache（缓存）

```
npm config set cache your node_cache path
npm config set prefix your nodejs ROOT path
```
